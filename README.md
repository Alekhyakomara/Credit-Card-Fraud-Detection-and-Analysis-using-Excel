# 💳 Credit Card Fraud Detection & Analysis using Excel

## 📌 Project Overview

This project analyzes credit card transaction data using Microsoft Excel to identify fraud patterns, measure financial risk, and generate actionable business insights through interactive dashboards and visualizations.

---

## 🎯 Objectives

- Detect fraudulent transaction patterns.
- Analyze fraud distribution across locations and transaction channels.
- Evaluate merchant category risk.
- Measure fraud exposure and financial impact.
- Build an interactive Excel dashboard.
- Recommend fraud prevention strategies.

---

## 📂 Dataset Information

The dataset contains **1,000 credit card transactions** with:

- Transaction ID
- Customer ID
- Date
- Location
- Transaction Type
- Card Type
- Merchant Category
- Transaction Amount
- Fraud Status

## 💡 Key Insights

- Fraud Rate: **16.4%**
- Legitimate Transactions: **836**
- Fraudulent Transactions: **164**
- Online transactions showed the highest fraud risk.
- Fraudulent transactions averaged **₹73,980**.
- Grocery and Electronics categories recorded higher fraud rates.
